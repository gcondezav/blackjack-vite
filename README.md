A modern Blackjack game built with Vite + JavaScript/TypeScript.

👉 Live Demo: https://gcv-blackjack-vite.netlify.app/

![Blackjack Preview](./preview.png)

⚙️ Technologies
Vite
JavaScript / TypeScript
HTML5
CSS3

🎮 How to Play
You receive 2 cards.
The goal is to reach 21 without going over.
You can ask for a card or stand.
The dealer plays next.

git clone https://github.com/gcondezav/blackjack-vite
cd blackjack-vite
npm install
npm run dev
